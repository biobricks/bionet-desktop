# BioNet Desktop Client
In Development. A React Desktop Client that interfaces with the [OpenFoundry API](https://gitlab.com/koeng/openfoundry-js).

## Install
In your terminal:
```
git clone https://github.com/matthewstewart/bionet-desktop.git && cd bionet-desktop
```

### Install Modules

```
sudo npm install --unsafe-perm=true --allow-root
```

### Run
```
npm start
```
